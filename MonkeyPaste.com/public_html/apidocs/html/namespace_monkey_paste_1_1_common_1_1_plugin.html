enum MonkeyPaste.Common.Plugin.MpContentQueryPropertyPathType

Data about a particular clip, its source and usage information.

Enumerator
None 
ClipText 

The plain text representation of a clip. Images are a base64 encoded string. FileLists are each path separated by line breaks (based on environment)

RawClipData 

Only meaningful for Text which is the actual HTML fragment of the clip. Result is the same as ClipText for Image or FileList.

ClipType 

The type of clip (Text,Image,FileList)

Title 

The title for a clip.

SourceAppPath 

A source process a clip was copied from

Note: This is generally reliable but not always accurate!

SourceAppName 

The source application name a clip was copied from.

SourceUrl 

A source web page a clip was copied from

Note: This information is not always available and dependant on how clipboard-friendly the source browser application is.

SourceUrlTitle 

The title of the web page a clip was copied from.

SourceUrlDomain 

A convienence property providing the domain from SourceUrl

CopyDateTime 

The last datetime a clip was copied.

CopyCount 

The amount of times this exact clip has been copied, disregarding any editing that may have occured along its history.

Note: This info may not be available depending on users preferences.

PasteCount 

The amount of times a clip has been pasted or dropped

Note: This info may not be available depending on users preferences or when pasted from a context menu.

SourceDeviceName 

A machine name the clip was copied from.

SourceDeviceType 

The type of machine a clip was copied from (will obviously only be windows for now)

LastOutput 

Only available in action sequences. Will have no affect if referenced in a MpParameterFormat

enum MonkeyPaste.Common.Plugin.MpParameterValueUnitType

The intended format for the MpParameterFormat.value or MpParameterFormat.values. See specific types for details. Many combinations of MpParameterValueUnitType and MpParameterControlType will be ignored.

Enumerator
None 
Bool 

A true or false value.

Integer 

A counting number. Used in conjunction with MpParameterControlType.Slider for integer sliders.

Decimal 

A double-precision number. Used in conjunction with MpParameterControlType.Slider for decimal sliders.

PlainText 

A plain text string.

Date 

Used in conjunction with MpParameterControlType.DateTimePicker for a date selector.

Time 

Used in conjunction with MpParameterControlType.DateTimePicker for a time selector.

FileSystemPath 

Used internally. Not relavant for any controls.

PlainTextContentQuery 

A string substitution value replacing "{query_key_name_here>}" with the query result from some clip as plain text.
Possible query key names are all MpContentQueryPropertyPathType keys.

Example: When paired with MpParameterControlType.TextBox the text box is given a query-field selector in the UI to simplify creating a content query. A useful hidden parameter may be a MpParameterControlType.TextBox with MpParameterValueUnitType.PlainTextContentQuery and setting the default value to '{ClipText}' to then always receive the plain text of the input clip.

Note: Text outside of curly braces or curly braces without a matching MpContentQueryPropertyPathType will be treated as plain text.

UriEscapedPlainTextContentQuery 

This is a convienence type converting that Html to plain text and then Uri encoding it. See PlainTextContentQuery for more info about content queries.

RawDataContentQuery 

/// Clips raw content is internally stored as follows:
Text: Html fragments (blocks and inlines)
Images: Base64 strings of the images bitmap
Files: A string of file system paths separated by environment new lines in plain text

See PlainTextContentQuery for more information about content queries

DelimitedPlainText 

Used with MpParameterControlType.MultiSelectList and MpParameterControlType.EditableList to receive a list of the selected MpParameterValueFormat.value's.

To access the values as a list of strings use the extension method MpPluginExtensions.GetRequestParamStringListValue(MpParameterMessageRequestFormat, string) on the MpMessageRequestFormatBase parameter.

DelimitedPlainTextContentQuery 

Gives MpParameterControlType.EditableList entries individual content-query field selectors.

See PlainTextContentQuery and DelimitedPlainText for more info.

CollectionComponentId

Used internally for action parameters.

Used internally for action parameters.

Used internally for action parameters.

enum MonkeyPaste.Common.Plugin.MpPluginType

The type of MpMessageRequestFormatBase request and MpMessageResponseFormatBase response this plugin is intended to handle.

Enumerator
None 
Clipboard 

Provides custom read and/or write support for any specified data format(s) during clipboard and drag-and-drop operations.

Analyzer 

Generates or annotates content given a set of established parameters that may include a single clip's content (in some particular format), meta-information about a clip along with a set of user-specified options based on parameters defined from MpParameterFormat items.

Fetcher 

(Experimental) Fetches a set of key/paramValue pairs (currently as MpIContact objects) for use with the 'Contact' type text templates available from the text editor toolbar button
