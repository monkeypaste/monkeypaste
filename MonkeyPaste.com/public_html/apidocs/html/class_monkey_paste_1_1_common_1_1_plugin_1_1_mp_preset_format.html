Detailed Description

A pre-defined preset allowing you to provided a unique name and/or icon for the preset and specific values for some or all plugin parameters. Any parameters not defined by the preset will use the parameters default value specified in the MpParameterFormat. Pre-defined presets cannot be deleted but can be reset back to its original state.
